# SuportAgent

Agente construído com **Microsoft Agent Framework (MAF)** para suporte ao cliente, utilizando **ChatHistoryProvider** para manter o contexto das conversas.

## Visão Geral

O **SuportAgent** foi desenvolvido para oferecer atendimento automatizado e contextualizado.  
Ele utiliza o histórico de conversas para fornecer respostas mais precisas e consistentes, garantindo uma experiência contínua para o usuário.

## Funcionalidades

- **Suporte automatizado**: responde dúvidas comuns de clientes.
- **Contexto persistente**: mantém histórico de interações com `ChatHistoryProvider`.
- **Integração com MAF**: aproveita middlewares e extensões do framework.
- **Logs de conversas**: registra interações para análise posterior.
- **Pipeline flexível**: pode ser integrado com outros agentes de suporte.
